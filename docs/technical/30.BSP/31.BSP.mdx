## STM32MP157F-DK2

To start building a *DK2* image run the following command:

```bash
make build dk2
```

Once the process is completed the full image is located in

```bash
shared/images/dk2/FlashLayout_sdcard_stm32mp157f-dk2-optee.raw
```

## 8MMINILPD4-EVKB

To start building a *8MMINILPD4-EVKB* image run the following command:

```bash
make build imx8mm-evk
```

Once the process is completed the full image is located in

```bash
shared/images/imx8mm-evk/core-image-minimal-imx8mm-lpddr4-evk.wic
```

## Qemu

There are two possible targets

- `qemu`: for ARMv7
- `qemu8`: for ARMv8

While all examples will concentrate on ARMv7, the other targets are available too.

### Qemu (ARMv7)

```bash
make build qemu
```

### Qemu (ARMv8)

```bash
make build qemu8
```
